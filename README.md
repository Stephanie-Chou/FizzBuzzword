FizzBuzzword
============
# Buzzwords

Submit your LinkedIn profile. This will search your profile for all the overused buzzwords of recent years and FizzBuzz it out.

Fizz: 'I'
Buzzword: If it is a buzzword
FizzBuzzword: If it is 'I am buzzword'
